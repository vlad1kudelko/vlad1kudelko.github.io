# Expo: платформа для React Native разработки

Expo — платформа для разработки React Native приложений с инструментами, упрощающими весь процесс.

## Установка

```bash
npm install -g expo-cli
npx create-expo-app my-app
cd my-app
npm start
```

## Expo SDK

```bash
npx expo install expo-camera expo-location expo-notifications
```

### Камера

```tsx
import { CameraView, useCameraPermissions } from 'expo-camera';

function App() {
  const [permission, requestPermission] = useCameraPermissions();
  
  if (!permission) return <View />;
  
  return (
    <CameraView style={{ flex: 1 }} />
  );
}
```

### Location

```tsx
import * as Location from 'expo-location';

async function getLocation() {
  const { status } = await Location.requestForegroundPermissionsAsync();
  if (status !== 'granted') return;
  
  const location = await Location.getCurrentPositionAsync({});
  return location.coords;
}
```

### Notifications

```tsx
import * as Notifications from 'expo-notifications';

await Notifications.scheduleNotificationAsync({
  content: { title: 'Title', body: 'Body' },
  trigger: null,
});
```

## EAS

```bash
# Установка
npm install -g eas-cli

# Login
eas login

# Build
eas build -p ios
eas build -p android
```

### EAS Update

```bash
# Публикация
eas update --branch production

# Автоматически через CI
eas update --branch production --message "Fix bug"
```

## Expo Go

```bash
# Запуск в Expo Go
npx expo start

# Туннель для удалённого доступа
npx expo start --tunnel
```
